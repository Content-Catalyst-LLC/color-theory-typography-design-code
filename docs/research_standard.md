# Research and Studio-Lab Standard

This repository supports interpretive and creative work. Code should help organize concepts, generate examples, compare forms, document sources, and support future web tools.

## Guardrails

- Do not reduce beauty, symbolism, music, design, or interpretation to automated scoring.
- Use generated datasets only as teaching materials unless documented otherwise.
- Preserve context, uncertainty, cultural care, and interpretive judgment.
- Keep article folders distinct from article maps.
- Use sibling article map names only as map links, not as article titles.
